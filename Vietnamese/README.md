## 📘 Ngày 1

### Chào Mừng

**Chúc mừng** bạn đã quyết định tham gia thử thách _30 ngày học Python_. Trong thử thách này, bạn sẽ học tất cả những gì cần thiết để trở thành lập trình viên Python và hiểu các khái niệm lập trình. Sau khi hoàn thành, bạn sẽ nhận được chứng chỉ _30DaysOfPython_.

### Giới Thiệu

Python là ngôn ngữ lập trình mức cao dành cho lập trình đa mục đích. Nó là ngôn ngữ mã nguồn mở, được giải thích, hướng đối tượng. Python được tạo bởi lập trình viên Hà Lan Guido van Rossum. Tên Python được lấy từ bộ phim hài nổi tiếng _Monty Python's Flying Circus_. Phiên bản đầu tiên được phát hành vào 20 tháng 2 năm 1991.

Thử thách 30 ngày này sẽ giúp bạn học phiên bản Python mới nhất (Python 3) từng bước một. Các chủ đề được chia thành 30 ngày, mỗi ngày có nhiều chủ đề với giải thích dễ hiểu, ví dụ thực tế và nhiều bài tập thực hành.

### Tại Sao Python?

- Python rất gần với ngôn ngữ con người, dễ học và sử dụng
- Python được sử dụng bởi nhiều công ty (Google, Netflix, Instagram, ...)
- Python được sử dụng cho: web apps, desktop apps, system admin, machine learning
- Python được yêu thích trong cộng đồng data science và machine learning
- Python đang "ăn" thế giới và bạn cần bắt kịp nó

### Cài Đặt Python

**Trên Windows:**
1. Vào https://www.python.org/
2. Bấm nút tải xuống phiên bản Windows
3. Chạy file cài đặt
4. ✓ Chọn "Add Python to PATH" khi cài đặt

**Trên macOS:**
1. Vào https://www.python.org/
2. Bấm nút tải xuống phiên bản macOS
3. Chạy file cài đặt

**Kiểm tra cài đặt:**

```bash
python3 --version
```

### Python Shell (Interactive Shell)

Python Shell cho phép bạn thực thi các lệnh Python từng dòng một.

Mở Terminal/Command Prompt và nhập:

```bash
python3
```

Sau đó bạn sẽ thấy dấu nhắc `>>>`. Gõ lệnh Python và bấm Enter.

Để thoát, nhập:

```python
exit()
```

### Các Kiểu Dữ Liệu Cơ Bản

**Số (Numbers):**
- Integer: -3, -2, -1, 0, 1, 2, 3
- Float: -3.5, -2.25, 0.0, 1.1, 2.2, 3.5
- Complex: 1 + j, 2 + 4j

**Chuỗi (String):**
- 'Xin chào'
- "Python là tuyệt vời"
- """Đây là chuỗi nhiều dòng"""

**Boolean:**
- True (đúng)
- False (sai)

**Danh Sách (List):**
```python
[1, 2, 3, 4, 5]
['Chuối', 'Cam', 'Xoài']
[1, 'Xin chào', 3.14, True]
```

**Từ Điển (Dictionary):**
```python
{
  'tên': 'Nguyễn Văn A',
  'tuổi': 25,
  'quốc_gia': 'Việt Nam'
}
```

**Bộ Giá Trị (Tuple):**
```python
('Nguyễn', 'Trần', 'Phạm')
(1, 2, 3, 4, 5)
```

**Tập Hợp (Set):**
```python
{1, 2, 3, 4}
{'Python', 'Java', 'JavaScript'}
```

### Kiểm Tra Kiểu Dữ Liệu

Sử dụng hàm `type()`:

```python
print(type(10))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type('Xin chào'))  # <class 'str'>
print(type([1, 2, 3]))   # <class 'list'>
print(type((1, 2)))      # <class 'tuple'>
print(type({1, 2, 3}))   # <class 'set'>
print(type({'key': 'value'}))  # <class 'dict'>
```

### Bình Luận (Comments)

Bình luận bắt đầu bằng ký tự `#` và được bỏ qua khi chạy code:

```python
# Đây là bình luận một dòng
print('Xin chào')  # Bình luận sau code

"""
Đây là bình luận nhiều dòng
Bạn có thể viết trên nhiều dòng
"""
```

### Lập Trình Cơ Bản

**Ví dụ - File helloworld.py:**

```python
# Ngày 1 - Thử thách 30 ngày Python

# Các phép toán
print(2 + 3)             # Cộng: 5
print(3 - 1)             # Trừ: 2
print(2 * 3)             # Nhân: 6
print(3 / 2)             # Chia: 1.5
print(3 ** 2)            # Lũy thừa: 9
print(3 % 2)             # Chia lấy dư: 1
print(3 // 2)            # Chia lấy phần nguyên: 1

# Kiểm tra kiểu dữ liệu
print(type(10))
print(type(3.14))
print(type('Xin chào'))
print(type([1, 2, 3]))
print(type({'tên': 'Nguyễn'}))
```

Chạy file:

```bash
python3 helloworld.py
```
